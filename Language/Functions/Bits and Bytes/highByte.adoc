= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Extracts the high-order (leftmost) byte of a word (or the second lowest byte of a larger data type).
[%hardbreaks]


[float]
=== Синтаксис
`highByte(x)`


[float]
=== Параметри
`x`: a value of any type


[float]
=== Повертає
Data type: `byte`.
