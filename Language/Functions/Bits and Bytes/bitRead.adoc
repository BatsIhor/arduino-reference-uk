= bitRead()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Reads a bit of a number.
[%hardbreaks]


[float]
=== Синтаксис
`bitRead(x, n)`


[float]
=== Параметри
`x`: the number from which to read. +
`n`: which bit to read, starting at 0 for the least-significant (rightmost) bit.


[float]
=== Повертає
The value of the bit (0 or 1).
