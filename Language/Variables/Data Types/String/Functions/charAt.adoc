= charAt()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Access a particular character of the String.

[%hardbreaks]


[float]
=== Синтаксис
`myString.charAt(n)`


[float]
=== Параметри
`myString`: a variable of type `String`. +
`n`: a variable. Очікує тип даних: `unsigned int`.


[float]
=== Повертає
The n'th character of the String.
