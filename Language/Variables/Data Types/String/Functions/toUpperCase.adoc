= toUpperCase()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Get an upper-case version of a String. As of 1.0, toUpperCase() modifies the String in place rather than returning a new one.
[%hardbreaks]


[float]
=== Синтаксис
`myString.toUpperCase()`


[float]
=== Параметри
`myString`: a variable of type `String`.


[float]
=== Повертає
