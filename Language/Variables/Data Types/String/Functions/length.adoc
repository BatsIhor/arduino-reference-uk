= length()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Returns the length of the String, in characters. (Note that this doesn't include a trailing null character.)

[%hardbreaks]


[float]
=== Синтаксис
`myString.length()`


[float]
=== Параметри
`myString`: a variable of type `String`.


[float]
=== Повертає
The length of the String in characters.
