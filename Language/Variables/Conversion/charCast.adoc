= char()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Converts a value to the `link:../../data-types/char[char]` data type.
[%hardbreaks]


[float]
=== Синтаксис
`char(x)` +
`(char)x` (C-style type conversion)


[float]
=== Параметри
`x`: a value. Allowed data types: any type.


[float]
=== Повертає
Data type: link:../../data-types/char[`char`].
