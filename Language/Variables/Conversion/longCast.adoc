= long()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Converts a value to the `link:../../data-types/long[long]` data type.
[%hardbreaks]


[float]
=== Синтаксис
`long(x)` +
`(long)x` (C-style type conversion)


[float]
=== Параметри
`x`: a value. Очікує тип даних: any type.


[float]
=== Повертає
Тип даних: link:../../data-types/long[`long`].
