= byte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Converts a value to the `link:../../data-types/byte[byte]` data type.
[%hardbreaks]


[float]
=== Синтаксис
`byte(x)` +
`(byte)x` (C-style type conversion)


[float]
=== Параметри
`x`: a value. Очікує тип даних: any type.


[float]
=== Повертає
Тип даних: link:../../data-types/byte[`byte`].
