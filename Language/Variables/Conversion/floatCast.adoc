= float()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Converts a value to the `link:../../data-types/float[float]` data type.
[%hardbreaks]


[float]
=== Синтаксис
`float(x)` +
`(float)x` (C-style type conversion)


[float]
=== Параметри
`x`: a value. Очікує тип даних: any type.


[float]
=== Повертає
Тип даних: link:../../data-types/float[`float`].

--
